# Ambers House Bed & Breakfast

This is the repository for a bookings and reservation project from tsawler's Go course on Udemy
- Built in Go 1.21
- Use the [Chi Router v5](https://github.com/go-chi/chi/v5)
- Uses [SCS Session Management](https://github.com/alexedwards/scs/v2)
- Uses [NoSurf](https://github.com/justinas/nosurf)
